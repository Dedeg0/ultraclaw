# `ultraclaw setup`

Initialize `~/.ultraclaw/ultraclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
ultraclaw setup
ultraclaw setup --workspace ~/.ultraclaw/workspace
```

To run the wizard via setup:

```bash
ultraclaw setup --wizard
```
